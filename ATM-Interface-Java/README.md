# CodSoft Task 3 - ATM Interface

## Project Overview
This project is a Java Swing based ATM Interface created as part of the CodSoft Java Programming Internship.

## Features
- ATM Machine class/interface
- Bank Account class to store balance
- Deposit amount
- Withdraw amount
- Check account balance
- Input validation
- Insufficient balance handling
- Interactive Java Swing GUI
- Success and failure messages

## Technologies Used
- Java
- Java Swing
- OOP Concepts
- Event Handling

## Folder Structure
```text
ATM-Interface-Java/
├── src/
│   └── ATMInterface.java
└── README.md
```

## Compile
```bash
javac src\ATMInterface.java
```

```bash
java -cp src ATMInterface
```
